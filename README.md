# salesforceSkymoto

## Conectar Salesforce

Este repositorio incluye un script para facilitar la conexión a una org de Salesforce usando Salesforce CLI.

### Requisitos

- Tener instalado Salesforce CLI (`sf`)

### Uso

```bash
./connect-salesforce.sh
```

Opcionalmente puedes indicar alias e instance URL:

```bash
./connect-salesforce.sh mi-org https://login.salesforce.com
```

Para ayuda:

```bash
./connect-salesforce.sh --help
```
